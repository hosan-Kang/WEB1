지금까지 배운 태그정리


strong => 볼드체로 텍스트 강조


u /u => 텍스트 밑줄 긋기


h 1-6 => 제목으로 설정하기(숫자크기에 따라 크기 달라짐. h는 기본적으로 약간의 굵기 있음)


br => 줄바꿈 코드. 한줄을 깔끔하게 줄바꿈 할 수 있다.
몇번이고 연속해서 쓸 수 있고, 굳이 닫지 않아도 되는것이 장점.


p /p => 단락 설정 코드.

p를 쓰면 좋은점은 HTML 코드만 보고 이것이 단락이라는 정보를 줄 수 있다는 점이고,


css를 이용해서 정교한 단락 공백 설정이 가능하다는 점이 좋다. 단점은 마지막에 태그를 닫아주어야 한다는 점.


img => 이미지 추가할 수 있는 태그. 그러나 단순히 태그만으로는 정보를 표시할 수 없어 속성을 달고 나와야 한다.
이미지 소스를 불러오기 위해서는 source 라고 쓰고 src라고 축약되는 속성을 쓰면되는데, "파일명.확장자"의 형태를 src 뒤에 덧붙여 주면 된다.
불러온 이미지의 크기를 조정하기 위해서는 width를 쓰면 됨. 픽셀단위로 "450"과 같이 수정도 가능하지만 "~%"로 지정하면 자동으로 조절해준다.


li /li ul /ul => li와 ul은 부모관계 처럼 항상 붙어 다녀야 하는것임 li 가 목차를 만들때 쓴다면, ul는 전체 범위를 할당.


ol /ol => 자동으로 번호 달아주는 시스템!


title => 웹 탭에 제목을 정해줌!


head => title 이나 utf-8 같은 문서를 읽는 방식에 대하여 설명해 주는 부분, 웹 전체의 전제 및 설명칸


body => 본문을 설명해주는 칸. 모든 html 문서는 잘 보면 head 와 body로 나누어진다!


html => html 임을 나타내는 태그
